# Streaks For YNAB

A web app for tracking spending habits with YNAB.

Built on [SvelteKit](https://svelte.dev/docs/kit/introduction).

For any bug reports or feature requests, visit the [issues](https://github.com/danielh-official/streaks-for-ynab-web-app/issues) page.

## Local Development

1. Clone: `git clone https://github.com/danielh-official/streaks-for-ynab-web-app.git`
2. Install packages: `pnpm install`
3. Run the server: `pnpm dev`
